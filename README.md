## picgo-plugin-avif 
本插件支持从本地文件路径或 URL 读取图片、压缩图片并将其**转换为 avif 格式**，，提高图片压缩率，减少图片体积。
同时支持CLI和GUI两种方式。

## Install安装
### CLI模式
```bash
picgo add avif
```
## Usage使用方法
```bash
picgo use plugins
```
